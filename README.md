Древнегреческая игра Петтейя (Petteia), которая позже эволюционировала в римские Латрункули (Latrunculi).

https://petteia-web.web.app/ 
**игра только для двоих через поделится ссылкой

Как играть: 
1. Фигуры размещаются на первых двух рядах перед каждым игроком. Подбрасывается монета, чтобы определить, кто будет белым игроком. Белые ходят первыми.
2. Все фигуры движутся как ладья в шахматах — по прямой линии на столько пустых клеток, сколько выберет игрок. Фигуры не могут перепрыгивать через другие фигуры. Фигуры не могут двигаться по диагонали.
3. Захват происходит только тогда, когда фигура перемещается таким образом, что одна фигура противника оказывается зажатой между двумя фигурами атакующего игрока. Можно захватить более одной фигуры одновременно. Захваченная фигура немедленно удаляется с доски и остается вне доски до конца игры. Каждый раз, когда фигура совершает захват, она получает еще один ход.
4. Фигура может безопасно переместиться на клетку между двумя фигурами противника, не будучи захваченной.
5. Игра заканчивается, когда игрок теряет все фигуры.
